# ✨ Glam & Glow

A production-grade, full-stack e-commerce platform built using **Spring Boot**, **PostgreSQL**, and **React**, designed with enterprise software engineering practices.

This project is being developed as a portfolio-ready application demonstrating modern backend development, secure authentication, clean architecture, DevOps practices, and AI-powered shopping features.

---

# 🚀 Project Vision

Glam & Glow is more than a basic online shopping application.

It aims to provide:

- Secure user authentication
- Modern REST APIs
- Intelligent shopping assistance
- Community interaction
- Scalable architecture
- Production-ready deployment

The final application will demonstrate real-world software engineering practices suitable for Software Development Engineer (SDE) interviews.
